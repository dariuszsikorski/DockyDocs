    Getting Started

    What is Docky?

    Docky is a HTML5 Bootstrap based document template, with built-in helpful features for easy customization. It works on desktop and mobile devices through the web browser. All you need is edit template html file then publish it along with your project.

    See example document and template file.

    Set up

    1. Dupliacte template.html file and rename it to your project's name. i.e. my-documentation.html.

    2. Open my-documentation.html file in any html editor.

    3. Create documentation inside section tags.

    4. Use this documentation and example file as reference.

    5. Publish documentation. Docky needs your my-documentation.html and assets folder to run.

    Feature list

    • search form - quickly search for sentences through document.
    • cross browser - tested on common web browsers such as Internet Explorer, Firefox, Safari, Google Chrome and mobile devices.
    • 20 ready topbar styles and custom colors.
    • syntax highlighting
    • included 49 color themes for 120 recognised code languages
    • fully responsive - from 320x480 mobile to 1920x1280 desktops.
    • page navigation is generated automatically by reading your documentation.
    • specific Url for each section
    • additional components, icons, buttons and more.
    • presentation video and html files included

    Custom Navigation Bar Color

    Find <div class="... doc-navbar navbar-default"> and add style="background-color: CustomColor;".

    Use tools such as Color Cop or Coolors for easy color selection.
